# Pedii Frontend Expert
Ao gerar código ou design para o Pedii, siga rigorosamente estes padrões:

### 1. Identidade Visual
* **Fundo:** Sempre use o preto profundo (base do logo).
* **Texto:** Branco puro para legibilidade.
* **Acentos & CTAs:** Use o degradê verde-limão vibrante do logotipo (de #72E890 para #35C75A) para botões, notificações de status, links ativos e ícones-chave.
* **Ícones:** Use ícones da biblioteca `lucide-react`, priorizando os de estilo vazado e tecnológico.

### 2. Stack Tecnológica
* **LP:** React 19 + Vite + Tailwind CSS + Lucide Icons.

### 3. Foco na Conversão (Landing Page)
* **Copywriting:** Foco total no benefício para o lojista: "Venda mais pelo WhatsApp", "Sem erros de estoque", "Controle total".
* **Imagens:** Use gráficos minimalistas e tecnológicos para ilustrar o fluxo de conversão.
* **Mobile First:** A LP deve ser perfeita em dispositivos móveis.

### 4. Importação de Skills da Comunidade
Sempre que aplicável, use estas skills do repositório Antigravity Awesome Skills:
* `tailwind-layout-genius`: Para grids complexos.
* `framer-motion-master`: Para animações suaves na Hero Section.

### 5. Deploy
* **LP:** Deploy no Cloudflare Pages no projeto `pedii-home-lp`.
